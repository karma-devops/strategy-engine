How Secrets Work in PULS-R
  1. Wallets (ETH) — Ethereum addresses for withdrawals. Set a Main wallet + optional secondary/tertiary.
  2. HyperLiquid DEX API — HL trading keys (private key + account address). Global env key works for operator; per-engine keys stored encrypted in DB.
  3. AI Inference — LLM provider API keys (Ollama Cloud, OpenRouter, OpenAI, Anthropic). Powers Strategy Studio + Assistant.
Security: All keys are Fernet-encrypted at rest. Raw secrets never leave the server. Only masked previews (e.g. 0xA871...8078) are shown in the UI.

Setting Up Your ETH Wallet
  1. Open MetaMask → copy your wallet address (starts with 0x).
  2. Click "Add Wallet" below → paste address + label.
  3. Set priority: Main (withdrawals go here), Secondary, or Tertiary.
  4. Save. The masked address appears in the list.

Setting Up HyperLiquid API Keys
  1. Go to app.hyperliquid.xyz → connect wallet.
  2. Click address (top right) → API Wallet.
  3. Copy the API Wallet Address + generate API Private Key.
  4. Click "Add HL Account" below → paste both values.
  5. Set priority: Main (global) or Secondary/Tertiary per engine.
Note: The API wallet is separate from your main wallet. Never use your main wallet's private key.
Never commit private keys to git. Keys are encrypted at rest. If compromised, generate a new one in HL and update here immediately.

Setting Up AI Inference
  1. Get an API key from your provider (see cards below).
  2. Click "Add Provider" → select provider, paste key + URL + model.
  3. Save. Click "Test" to verify the key works.
